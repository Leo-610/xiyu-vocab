# 安全与合规说明

## 1. 隐私保护

### 1.1 采集范围（最小化原则）

| 数据项 | 用途 | 是否必需 |
|--------|------|----------|
| 微信 openid | 用户身份识别 | 是（上线后） |
| 昵称/头像 | 展示 | 可选 |
| 学习记录 | 进度同步、统计 | 是 |
| 手机号 | 无 | 否 |

### 1.2 用户协议要点（草案）

- 本应用为 **教育学习工具**，不向第三方出售用户数据。
- 学习数据仅用于个人进度统计与产品改进（ anonymized 聚合分析）。
- 用户可申请导出或删除个人数据（上线后提供入口）。

### 1.3 隐私政策要点（草案）

- 说明采集项、存储期限、安全措施。
- 小程序发布前需在公众平台配置《用户隐私保护指引》。
- 答辩阶段 H5 原型可使用本地存储（localStorage），不涉及云端个人信息。

## 2. 内容合规

### 2.1 词库与例句

- 由西班牙语专业同学编写与校对，确保语法与释义准确。
- 例句避免政治、宗教、暴力等敏感内容。
- DELE 真题语料使用需注明来源，遵守版权与合理使用原则。

### 2.2 配图版权

| 来源 | 要求 |
|------|------|
| 团队自绘 | 著作权归项目组，答辩/申报书中说明 |
| AI 生成 | 标注生成工具与日期，遵守平台 ToS |
| 免版权图库（Unsplash 等） | 保留作者信息与许可证记录 |
| 教材/网络图片 | **禁止**未授权商用 |

建议建立 `data/images/COPYRIGHT.md` 台账，逐图记录来源与授权。

## 3. 微信小程序审核

- **类目**：教育 → 在线教育 / 语言学习
- **初期策略**：无 UGC（用户不上传内容），仅只读词库，审核通过率更高
- **必备页面**：用户协议、隐私政策（可 web-view 或原生页）
- **主体**：建议通过 **学校创新创业学院** 或 **院系** 申请企业/组织主体小程序

## 4. 数据安全

- 生产环境全站 **HTTPS**
- 数据库账号最小权限，禁止 root 直连公网
- 定期备份（建议每日增量 + 每周全量）
- API 鉴权：JWT 或微信 session_key 校验
- 防 SQL 注入：参数化查询
- 防刷接口：登录用户 rate limit

## 5. 等保与备案（远期）

- 用户量较小时通常无需等保测评；若校内统一部署或用户过万，按学校网信部门要求办理。
- 使用国内云服务器需 **ICP 备案**（小程序云开发可简化部分流程）。

## 6. 答辩说明话术（建议）

> 本项目采用最小化数据采集策略，词库与配图由团队原创并建立版权台账；上线后将通过微信官方登录、HTTPS 传输与数据库加密备份保障用户学习数据安全，并配合学校主体完成小程序合规审核。
