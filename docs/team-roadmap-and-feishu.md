## 二、西语同学 A — 词库与语言内容

**角色定位**：DELE 分级词库主编、例句与变位质检。

### 2.1 任务清单（按优先级）

| 截止 | 任务 | 交付物 | 路径/标准 |
|:----:|------|--------|-----------|
| 第 2 周 | 熟悉模板与规范 | 签认已读 | 阅读 [`spanish-content-guide.md`](spanish-content-guide.md) |
| 第 3 周 | **A1 首批 100 词** | CSV | `data/batches/A1/words_A1.csv` |
| 第 5 周 | **A1 扩至 300 词** | CSV 增量 | 同上，含例句、4 干扰项 |
| 第 6 周 | **动词变位 JSON** | 至少 20 个 A1 动词 | 字段 `conjugation_json`，见 [`conjugation-schema.json`](conjugation-schema.json) |
| 第 8 周 | **易混词辨析文案** | 8 组 | `confusable_pairs.note_zh` 替换「待填写」 |
| 第 10 周 | **A2 首批 200 词** | CSV | `data/batches/A2/words_A2.csv` |
| 第 12 周 | 质检报告 | 1 页 | 飞书文档：错词/敏感词排查签字 |

### 2.2 每周例行（约 3–5 小时）

- [ ] 在飞书「词库进度表」更新本周新增词数
- [ ] 新词 CSV 行自检：lemma 无重音错误、meaning_zh 准确、例句无敏感内容
- [ ] 与技术同学确认：交付后 24h 内能在 App 里刷到新词

### 2.3 不要做的事

- 不改代码、不直接改数据库
- 不使用未授权 DELE 真题原文（可「参考分级」，注明自制）
- 例句避免政治/宗教/暴力内容

## 三、西语同学 B — 配图与调研

**角色定位**：视觉统一、版权合规、用户调研与大创用户数据。

### 3.1 任务清单

| 截止 | 任务 | 交付物 | 路径/标准 |
|:----:|------|--------|-----------|
| 第 2 周 | **风格样张 2 张** | hablar、casa | `data/images/A1/`，800×800，教师评审通过 |
| 第 4 周 | **A1 配图 30 张** | 与 CSV `image_file` 同名 | 同目录 + 更新 [`COPYRIGHT.md`](../data/images/COPYRIGHT.md) |
| 第 5 周 | **用户问卷定稿** | 10–15 题 | 飞书问卷 + `docs/survey/questionnaire.md` |
| 第 6–8 周 | **试用推广 + 回收** | ≥30 份有效问卷 | `docs/survey/results/` + 飞书表格导出 |
| 第 8 周 | **问卷分析报告** | 3–5 页 | `docs/survey/analysis.md`（痛点、满意度、建议） |
| 第 10 周 | A1 配图补全至 300 | 与 A 同学词表对齐 | 缺图清单在飞书表标记 |
| 第 12 周 | 配图版权包 | 汇总 | 版权台账 + 来源截图打包给指导教师 |

### 3.2 配图规范（摘要）

- 尺寸 **800×800**，统一风格（插画风 / 扁平，二选一）
- 文件名与 CSV 中 `image_file` **完全一致**（如 `hablar.jpg`）
- 每张图在 `COPYRIGHT.md` 登记：来源、作者、许可证、日期

### 3.3 调研建议题目（可放飞书问卷）

1. 你目前用什么工具背西语？最大痛点？
2. 对 DELE 分级词库的刚需程度（1–5）
3. 配图识记 vs 纯文字，你更倾向？（1–5）
4. 每日愿学新词数量（5/10/15/20）
5. 是否愿意试用本小程序 7 天？（留联系方式可选）

## 五、大创指导教师 — 任务清单

> 可直接转发 [`school-approval-checklist.md`](school-approval-checklist.md) 与 [`school-approval-application.docx`](school-approval-application.docx)。

### 5.1 行政与合规（高优先级）

| 事项 | 说明 | 建议时间 |
|------|------|----------|
| **确认学校主体** | 是否允许以院系/学校组织注册微信小程序 | 第 1 周 |
| **担任小程序管理员** | 微信公众平台扫码管理员（建议教师） | 第 2–3 周 |
| **审批/介绍信** | 创院/网信/法务：主体 + 备案 + 协议审阅 | 第 2–4 周 |
| **认证经费** | 微信认证约 300 元/年，明确出处 | 第 3 周 |
| **审阅隐私政策/用户协议** | 定稿后签字或邮件确认 | 上线前 |
| **中期/结题** | 审核中期报告、答辩出席、推荐省级 | 按院系节点 |

### 5.2 学术与内容把关

| 事项 | 说明 |
|------|------|
| DELE 分级合理性 | 抽检 A 同学词库分级与释义 |
| 内容合规 | 例句/配图无敏感、无「官方 DELE」误导宣传 |
| 配图风格 | 第 2 周评审 B 同学 2 张样张，定调后批量生产 |
| 大创书稿 | 指导摘要、创新点表述（西语+AI 可写「规划中」） |

### 5.3 教师无需亲自做

- 不写代码、不填 CSV
- 可不日常盯飞书表（学生负责人每周发 1 页进度即可）

## 六、飞书建群与协作（手把手）

### 6.1 建群与成员

1. 打开 **飞书** → 左侧 **消息** → 右上角 **+** → **创建群组**
2. 群名称建议：`西语背单词-大创项目组`
3. 群描述：`DELE背单词小程序 | 内容+技术 | 大创202x`
4. 添加成员：**指导教师、西语 A、西语 B、你（群主/管理员）**
5. 群设置 → **群管理**：
   - 开启 **仅群主/管理员可 @所有人**（避免骚扰教师）
   - 固定公告（见下方模板）

**群公告模板（复制粘贴）：**

```
【项目】西语背单词 · DELE 分级识记
【分工】A=词库/变位/易混词 | B=配图/调研 | 技术=开发/上线
【资料】飞书文件夹「西语背单词-共享资料」
【填表】词库进度表、配图台账、每周站会（链接见置顶）
【交付】A/B 只填表+交文件，不改代码；交付后 @技术 导入
【例会】每周 X 晚 20:00 线上 15 分钟（可选）
```

---

### 6.2 建共享文件夹（放文档与模板）

1. 飞书 → **云文档** → **新建** → **文件夹**
2. 命名：`西语背单词-共享资料`
3. 子文件夹建议：

```
西语背单词-共享资料/
├── 00-项目管理/          （进度表、会议纪要）
├── 01-词库模板/          （word_template.csv 副本）
├── 02-配图规范/          （样张、COPYRIGHT 模板）
├── 03-大创材料/          （立项书、中期、答辩 PPT）
├── 04-问卷与数据/        （回收结果、分析）
└── 05-合规/              （协议草案、审批扫描件）
```

4. 文件夹 → **分享** → 添加群成员 → 权限 **可编辑**（教师可设 **可阅读**）
5. 把 GitHub/本地 `docs/` 里关键 md、模板 **上传副本**（方便非技术同学不看代码）

---

### 6.3 建「词库进度表」（多维表格，核心）

1. 云文档 → **新建** → **多维表格**
2. 命名：`词库与配图进度`
3. 新建数据表 **「A1 词库」**，字段建议：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| lemma | 文本 | 西语词条 |
| 负责人 | 人员 | 默认同学 A |
| 状态 | 单选 | 未开始 / 编写中 / 已校对 / 已导入 App |
| DELE 等级 | 单选 | A1 |
| 是否有例句 | 复选框 | |
| 是否有变位 | 复选框 | |
| 配图文件名 | 文本 | 如 hablar.jpg |
| 配图状态 | 单选 | 缺图 / B 制作中 / 已完成 |
| 备注 | 文本 | |

4. 复制表结构建 **A2 / B1…** 或用一个表加「等级」字段
5. 另建数据表 **「配图版权台账」**：

| 字段 | 类型 |
|------|------|
| 文件名 | 文本 |
| 对应 lemma | 文本 |
| 来源 | 单选：自绘 / AI / 图库 |
| 许可证 | 文本 |
| 登记日期 | 日期 |
| 负责人 | 人员（同学 B） |

6. **分享**：获取链接 → 发到群置顶 → 设 **群内成员可编辑**

> **快捷方式**：直接使用 [`docs/feishu/`](../feishu/) 下 CSV **导入建表**（含示例行与字段说明），见 [`feishu/README.md`](../feishu/README.md)。

**同学 A/B 日常只改这张表**，不必懂 Git。

---

### 6.4 建「每周站会 / 阻塞问题」表

数据表 **「Weekly」** 字段：

| 字段 | 类型 |
|------|------|
| 周次 | 文本（如 2026-W23） |
| 填写人 | 人员 |
| 本周完成 | 文本 |
| 下周计划 | 文本 |
| 需要老师/技术帮助 | 文本 |
| 填写日期 | 日期 |

每周日 22:00 前三人各填一行；你汇总 1 段发教师（不必每次 @老师）。

---

### 6.5 建问卷（用户调研）

**方式 A — 飞书问卷（推荐，数据进飞书）**

1. 云文档 → **新建** → **问卷**（或 **收集** → 创建问卷）
2. 标题：`西语背单词小程序 用户需求调研`
3. 按第三节 B 同学题目添加单选/量表/填空
4. 设置 → **谁可以填写**：获得链接的任何人 或 仅校内（需登录）
5. **分享** → 复制链接/二维码 → 发到年级群、西语群
6. 回收后：**导出 Excel** → 存 `docs/survey/results/` + 上传飞书文件夹

**方式 B — 飞书多维表格「表单视图」**

1. 在「问卷与数据」文件夹新建多维表格
2. 视图 → **新建表单视图** → 把字段拖成对外表单
3. 适合与进度表同一套数据打通

---

### 6.6 群内置顶与机器人（可选）

- **置顶 3 条消息**：共享文件夹链接、进度表链接、问卷链接
- **日历**：创建重复事件「每周组会 15min」，邀请群成员
