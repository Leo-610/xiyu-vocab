# 飞书多维表格 CSV 导入包

本目录 CSV 可直接导入飞书 **多维表格**，建表后按下方说明调整字段类型即可。

---

## 一、导入步骤（每个 CSV 建一张表）

1. 打开飞书 → **云文档** → **新建** → **多维表格**
2. 进入表格 → 左上角表名旁 **⋯** → **导入** → **CSV 文件导入**  
   （或：新建多维表格时选择 **从 CSV 导入**）
3. 选择对应 CSV，编码选 **UTF-8**
4. 第一行作为**字段名**，确认导入
5. 按下方「字段类型对照」逐列改类型（导入后默认多为「文本」）

> 若中文乱码：用 Numbers / Excel 打开 CSV 确认 UTF-8，或用 VS Code 另存为 UTF-8 with BOM 后再导。

## 四、导入后推荐操作

1. **分享**：右上角分享 → 添加群「西语背单词-大创项目组」→ 可编辑  
2. **置顶**：把 Base 链接发到群公告  
3. **仪表盘**（可选）：统计「已导入App」词数、缺图数量  
4. **自动化**（可选）：状态改为「已校对」时 @技术负责人  

---

## 五、与 Git 仓库同步

| 飞书表 | 仓库路径 |
|--------|----------|
| 词库行（状态=已校对） | 导出 CSV → 合并进 `data/batches/A1/words_A1.csv` |
| 配图（配图状态=已完成） | 图片放 `data/images/A1/` |
| 版权台账 | 同步更新 `data/images/COPYRIGHT.md` |

技术同学导入命令：

```bash
node backend/src/seed.js data/batches/A1/words_A1.csv
```

导入成功后，在飞书把对应行 **状态** 改为 **已导入App**。

---

## 六、复制到新 Base 的快速顺序

1. 导入 `词库与配图进度.csv`  
2. 同 Base 内 **添加数据表** → 导入 `配图版权台账.csv`  
3. 添加 `每周站会.csv` → 为该表建 **表单视图**  
4. 添加 `大创里程碑.csv`  

总耗时约 **10 分钟**。
