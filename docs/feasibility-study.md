# 西语背单词小程序 — 可行性研究报告

## 1. 项目背景

西班牙语专业学生在日常学习与 DELE 备考中，普遍面临以下痛点：

- **通用工具西语支持弱**：百词斩、墨墨背单词等以英语为主，西语词库质量参差，缺乏动词变位、阴阳性等结构化信息。
- **纸质词书不便携**：携带与检索效率低，内容更新滞后。
- **语境与考纲脱节**：缺少与 DELE A1–C2 等级对齐、可溯源的例句与复习策略。

本项目旨在开发一款面向西语学习者的 **Image-based 背单词工具**（类百词斩模式），并规划 RAG/LLM 升级路线以冲击大学生创新创业训练计划（大创）国家级项目。

## 2. 市场与竞品分析

| 产品 | 优势 | 西语短板 |
|------|------|----------|
| 百词斩 | 图像联想、游戏化、用户基数大 | 西语词库有限，无 DELE 分级 |
| 墨墨背单词 | SM-2 算法成熟 | 西语例句与变位支持不足 |
| Quizlet | 社区卡组丰富 | 内容质量不可控，国内访问与合规问题 |
| Anki | 高度可定制 | 上手门槛高，无官方西语 DELE 体系 |

**差异化定位**：DELE 分级词库 + 配图识记 + 西语专属字段（变位/阴阳性/易混词）+ 后续 RAG 语境例句。

## 3. 目标用户

- 高校西班牙语专业本科生（核心用户）
- DELE 备考者（A1–C2）
- 西语爱好者与二外学习者

## 4. 功能范围

### 4.1 一期（立项答辩 / MVP）

- 每日新词包（默认 10 词）
- 四选一识记（西语 → 中文/图片）
- 错题本
- 学习统计（打卡、进度）
- H5 可交互原型演示

### 4.2 二期（立项后 1–2 月）

- 微信小程序上线
- 5000+ 词库与 CDN 配图
- 微信登录与用户进度云同步
- SM-2 间隔重复复习

### 4.3 三期（大创升级）

- RAG：基于 DELE 语料的可溯源例句
- LLM：错题解析、易混词对比（人工审核闭环）
- 可视化：掌握热力图、遗忘曲线、等级雷达图

## 5. 技术可行性

| 模块 | 方案 | 成熟度 |
|------|------|--------|
| 前端 | uni-app Vue3 → H5 / 小程序 | 高 |
| 后端 | 微信云开发 或 Node.js + 云服务器 | 高 |
| 数据库 | MySQL 8 | 高 |
| 图片存储 | 腾讯云 COS / 阿里云 OSS | 高 |
| RAG | pgvector + Embedding + LLM API | 中（二期） |

团队配置（3 人）：2 名西语生负责内容与质检，1 名技术负责架构与开发，符合小团队敏捷交付能力。

## 6. 经济可行性

- **开发成本**：以开源技术栈为主，云开发/学生优惠云资源，初期可控制在千元级/年。
- **内容成本**：词库与配图由团队自制，无版权采购压力（需规范 AI/图库授权）。
- **收益模式**（远期）：免费基础版 + DELE 专题包 / 院校合作授权（立项阶段不涉及商业化）。

## 7. 运营与推广

- 校内西语专业试用（目标 30+ 问卷样本）
- 西语系老师推荐与 DELE 辅导班合作
- 连续打卡与错题本驱动留存

## 8. 风险与对策

| 风险 | 对策 |
|------|------|
| 无小程序 AppID | 答辩用 H5；同步申请学校主体 |
| 5000 词产能 | 分 DELE 等级批次；CSV 模板 + 质检 |
| LLM 幻觉 | RAG 检索增强 + 人工审核 |
| 答辩周期紧 | 50 词 demo + 架构与创新点 |

## 9. 结论

项目在 **需求真实性、技术可实现性、团队分工合理性** 三方面均可行。建议以 H5 原型完成立项答辩，立项后 2 个月内交付微信小程序 MVP，并分阶段引入 RAG/LLM 以支撑省级乃至国家级大创申报。
