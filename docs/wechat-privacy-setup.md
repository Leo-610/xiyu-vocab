# 微信小程序 · 用户隐私保护指引（必配）

控制台报错 `chooseAvatar:fail api scope is not declared in the privacy agreement`  
表示：**公众平台尚未声明相关隐私接口**。代码已做授权适配与降级，但仍须完成下方配置。

## 操作路径

1. 打开 [微信公众平台](https://mp.weixin.qq.com/) → 小程序「西语背单词」
2. **设置 → 基本设置 → 服务内容声明 / 用户隐私保护指引**
3. 按向导「更新」隐私保护指引

## 建议勾选的采集项

| 信息类型 | 用途说明（可照抄） | 对应能力 |
|----------|-------------------|----------|
| 微信昵称 | 展示学习主页昵称（用户自愿填写） | 昵称输入 |
| 微信头像 | 展示学习主页头像（用户自愿选择） | `chooseAvatar` / 选图 |
| OpenID / 用户标识 | 登录识别与学习进度同步 | `code2Session` |
| 选中的照片或视频文件 | 用户从相册选择头像时读取 | `chooseImage` |

**不采集**：手机号、位置、通讯录、通讯记录等。

## 隐私政策链接

可填 App 内页（需已配置业务域名，或使用 H5）：

`https://bjtu.club/#/pages/legal/privacy`

或先用公众平台默认指引文案，保存后重新编译预览。

## 验证

1. 保存指引后等待约 1–5 分钟  
2. 开发者工具重新编译 → 清除缓存 → 打开小程序  
3. 同意隐私页 → 登录 → 完善资料选头像，不应再报 `privacy agreement`
