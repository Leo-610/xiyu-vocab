## 七、提交微信审核前自检

- [ ] AppID 已填入 `frontend/src/manifest.json` → `mp-weixin.appid`
- [ ] 生产 API 可访问，域名已加入公众平台白名单
- [ ] 隐私政策联系方式已填写真实院系邮箱
- [ ] 审核路径可走通：首页 → 学习 → 错题 → 统计
- [ ] 体验版二维码供指导教师真机验收
- [ ] 审核说明已按 [上线指南第九章](wechat-miniprogram-launch-guide.md#九审核说明模板可直接改) 填写

## 八、建议时间线（供审批排期）

| 周次 | 事项 |
|------|------|
| 第 1 周 | 院系立项确认 + 主体申请意向 |
| 第 2–3 周 | 校内法务/网信材料 + 域名备案启动 |
| 第 4 周 | 微信注册、认证、小程序备案 |
| 第 5 周 | 生产部署、体验版、教师验收 |
| 第 6 周 | 提交审核、发布正式版 |
