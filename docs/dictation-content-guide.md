# 听写模式 · 内容组分工（第一档）

> **功能说明**：用户听发音 → 输入西班牙语拼写 → 系统判对错 → 错词进入错题本。  
> **技术已实现**：`GET /api/words/dictation`、`POST /api/words/dictation/check`、App 页「听写练习」。  
> **判题规则（当前）**：不区分大小写；**重音/ñ 写错仍算对**（A1 友好模式，后期可改严格模式）。

---

## 同学 A — 语言与词表（主责）

### 1. 划定「听写试点词单」（优先 A1，20～100 词）

在飞书表「词库进度」中新增列 **`dictation_pilot`**（填 `是` / `否`），或单独 sheet「听写试点」：

| 字段 | 说明 |
|------|------|
| word_id / lemma | 与 CSV 一致 |
| level | A1 优先 |
| 是否纳入听写 | 是 |
| 备注 | 如：易混、需加干扰说明 |

**选词原则**

- 优先：**高频、拼写规则清晰**的名词/形容词/动词原形  
- 暂缓：同一词多种拼写变体、缩写、专有名词、过长短语  
- 动词：试点阶段只用 **原形**（infinitivo），变位听写留到第二档

### 2. 核对 `lemma` 拼写（听写答案的唯一标准）

- 逐词对照 RAE / 教材，确认 CSV 中 `lemma` 为听写**唯一标准答案**  
- 若存在合理变体（如 `adiós` / `adios`），在备注写明；技术端可后续加「别名列」

### 3. 补充 IPA（`ipa` 列）

- 答错/答对后 App 会展示音标，请保证与 DELE A1 常用标注一致  
- 无 IPA 可暂留空，但不利于答辩展示

### 4. 标注「可接受拼写别名」（可选，第二期）

若需要严格区分重音，在备注记录；当前系统默认**宽松判题**，答辩时可说明「A1 先练拼写骨架，B 级再开严格重音」。

### 5. 听写专用 QC 清单（交付前自检）

对试点词逐条打勾：

- [ ] `lemma` 无多余空格、无中文标点  
- [ ] 中文释义 `meaning_zh` 与 lemma 唯一对应  
- [ ] 例句 `example_es` 中该词拼写与 lemma 一致  
- [ ] 无「一词多写法未说明」的情况  

**交付路径**：更新 `data/batches/A1/words_*.csv` 或飞书导出 → @技术同学 导入。

## 同学 B — 发音与素材（主责）

### 1. 发音来源二选一（试点词必须至少有一种）

| 方案 | 做法 | 推荐场景 |
|------|------|----------|
| **A. 录制 mp3** | 西语母语/标准朗读，导出 mp3 | 答辩演示、小程序（无 Web Speech） |
| **B. 暂用 TTS** | 不录 mp3，H5 用浏览器朗读 | 本地开发/demo 够用 |

**若选 A（推荐答辩用）**

- 文件命名与 CSV 列 `audio_file` 一致，如 `hola.mp3`  
- 存放：`data/audio/A1/`（或与技术约定 `static/audio/`）  
- CSV 填 `audio_url` 或 `audio_file`（与技术对齐模板）  
- 要求：清晰、无背景音乐、语速适中（A1 略慢）

### 2. 版权与来源登记

在 `data/images/COPYRIGHT.md` 同级增加 **`data/audio/COPYRIGHT.md`**：

| 文件名 | 朗读者 | 录制日期 | 是否可商用 | 备注 |
|--------|--------|----------|------------|------|

### 3. 试点批次建议

- **第一批 20 词**：与技术联调，确认 App「听写练习」能播、能判  
- **第二批扩至 100 词 A1**：覆盖答辩 demo  
- 每条词在飞书「词库进度」标记 **`audio_ready`**：是/否

### 4. 易混词听写说明（配合同学 A）

对 ser/estar、por/para 等，**不要**在同一听写组连续出现近音词，除非 A 已在备注写清辨析话术（供答辩用）。

## 两人共同完成

| 事项 | 分工 | 截止建议 |
|------|------|----------|
| 确定 20 词试点列表 | A 选词 + B 确认都能录/播 | 第 1 周 |
| 联调听写 | B 交 mp3，A 交 CSV，技术导入 | 第 2 周 |
| 答辩 demo 脚本 | 各准备 1 分钟：A 讲选词与判题规则，B 讲发音质量 | 答辩前 |

**联调检查（两人一起过一遍 App）**

1. 首页 → 「听写练习」  
2. 每词：能播放 → 输入正确拼写 → 显示「拼写正确」  
3. 故意输错 → 进入错题本 → 错题本可复习  
4. 故意少写重音（如 `como` 代替 `cómo`）→ 当前应判**对**（与 A 确认是否符合预期）
