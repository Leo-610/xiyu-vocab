# 西语同学内容制作指南

本文档面向 **词库主编（同学 A）** 与 **视觉质检（同学 B）**，说明如何填写词库、交付配图、以及答辩前 50 词任务包。

---

## 1. 你们的总体目标

| 阶段 | 词量 | 截止日期（建议） |
|------|------|------------------|
| 答辩 demo | **50 词** + 配图 | 答辩前 4 天 |
| MVP | 500–1000 词 | 立项后 1 月 |
| 完整词库 | **5000–6000 词** | 立项后 3–4 月 |

---

## 2. CSV 填表规范

### 2.1 文件位置

- 模板：[`data/word_template.csv`](../data/word_template.csv)
- 答辩样例（可参考）：[`data/sample_50.csv`](../data/sample_50.csv)
- 你们维护的文件建议命名：`data/words_A1_batch1.csv`

### 2.2 必填字段

| 列名 | 说明 | 示例 |
|------|------|------|
| lemma | 词典原形，小写 | hablar |
| pos | 词性缩写 | n / v / adj / adv / prep / pron / int |
| gender | 名词填 m/f，其他填 n/a | f |
| level | DELE 等级 | A1 |
| meaning_zh | 中文释义，简明 | 说话；讲 |
| distractor_1~3 | 三个干扰项（中文） | 吃饭 / 睡觉 / 写字 |
| example_es | 西语例句（短） | Hablo español. |
| example_zh | 例句中文 | 我说西班牙语。 |
| image_file | 图片文件名 | hablar.jpg |
| tags | 标签，逗号或竖线分隔 | 日常,变位 |
| copyright_note | 版权说明 | AI生成_2026-06 |

### 2.3 词性缩写对照

| 缩写 | 含义 |
|------|------|
| n | 名词 |
| v | 动词 |
| adj | 形容词 |
| adv | 副词 |
| prep | 介词 |
| pron | 代词 |
| int | 感叹词 |

### 2.4 干扰项设计原则

- 与正确答案**词性相同或相近**更佳
- 避免过于离谱的选项（一眼假）
- 易混词可作干扰项（如 ser / estar），但答辩 demo 不宜过多
- 三个干扰项之间不要重复

### 2.5 DELE 分级参考

- **A1**：基础日常（hola, casa, comer…）
- **A2**：简单交流（ciudad, por/para…）
- **B1–B2**：中级（抽象名词、复杂动词）
- **C1–C2**：高级（学术、书面语）

参考：《现代西班牙语》、DELE 官方大纲、塞万提斯学院词汇表。

## 3. 配图规范（同学 B 主责）

### 3.1 技术参数

- 尺寸：**800 × 800 px**
- 格式：JPG 或 PNG
- 命名：与 CSV 中 `image_file` 一致，如 `hablar.jpg`
- 存放：`data/images/`

### 3.2 风格统一

- 答辩 demo：建议 **扁平插画** 或 **统一风格 AI 出图**
- 避免：水印、低分辨率、与词义无关的网图
- 动词：可配动作场景；名词：配实物；形容词：配对比场景

### 3.3 版权记录

每批图片在 `data/images/COPYRIGHT.md` 记录：

```markdown
| 文件名 | 来源 | 授权 |
|--------|------|------|
| hablar.jpg | Midjourney 自生成 | 团队原创 |
```

## 4. 答辩前 50 词任务包

### 4.1 分工建议

**同学 A**（约 25 词）：
- A1 核心词 20 个：问候、代词、基础动词（ser/estar/tener/ir）
- A2 语法词 5 个：por, para, ciudad, país, español

**同学 B**（约 25 词）：
- A1 名词/形容词 20 个：casa, libro, agua, bueno, grande…
- 对应 50 张配图 + 交叉质检

### 4.2 50 词清单（按优先级）

详见 [`docs/spanish-50-word-checklist.md`](spanish-50-word-checklist.md)

### 4.3 交付检查

提交前请确认：
- [ ] CSV 能在 Excel 中正常打开（UTF-8）
- [ ] 每行 lemma、meaning_zh、三个干扰项非空
- [ ] level 仅为 A1/A2/B1/B2/C1/C2
- [ ] 每张 image_file 在 data/images/ 中有对应文件
- [ ] 例句语法经两人交叉校对

---

## 5. 质检 Checklist（每批词库）

### 5.1 语言准确性

- [ ] 释义准确、无歧义
- [ ] 例句语法正确、与等级匹配
- [ ] 动词使用正确时态/人称
- [ ] 名词阴阳性标注正确
- [ ] 拉美/西语差异：如有差异，在 tags 标注

### 5.2 教学合理性

- [ ] DELE 等级与词难度匹配
- [ ] 干扰项难度适中
- [ ] 易混词对已标注（tags 含「易混」）

### 5.3 合规

- [ ] 无政治/宗教/暴力敏感内容
- [ ] 配图版权已记录
- [ ] 例句非整段抄袭教材

## 6. 大创材料（同学 B 额外负责）

- **文献综述**：西语数字化学习、间隔重复、Image-based 记忆（2000 字）
- **用户调研**：问卷 30+ 份（同学院西语生）
- **竞品截图**：百词斩/墨墨西语词库对比

问卷样题：
1. 你目前用什么工具背西语单词？
2. 现有工具最大问题是什么？（变位/例句/考纲/便携性）
3. 是否愿意使用配图识记类 App？（1–5 分）

## 7. 提交方式

1. CSV 发群 / 提交至 `data/` 目录
2. 图片打包至 `data/images/`
3. @技术同学 运行导入：

```bash
python3 scripts/import_words.py --csv data/your_batch.csv --export-json frontend/src/static/words.json --dry-run
```

校验通过后技术同学合并进演示数据。

## 8. 常见问题

**Q：一个 lemma 多个词性怎么办？**  
A：分多行，如 `trabajo,n,m` 与 `trabajar,v,n/a`。

**Q：配图来不及怎么办？**  
A：答辩 demo 可先用占位图，但 50 词中至少 **30 张** 需有真实配图。

**Q：tags 写什么？**  
A：主题（日常/旅游）、语法（变位/易混）、专题（DELE-A2）。
